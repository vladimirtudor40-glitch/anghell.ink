ANGHELL.INK — WHERE TO PUT THE PHOTOS
=====================================

The site works right now with styled placeholders. To replace them with
Alina's real work, just drop image files with these exact names into the
folders below. They appear automatically on the next page load — no code
changes needed.

------------------------------------------------------------------
1) HERO + PORTRAIT  →  assets/img/
------------------------------------------------------------------
  hero.jpg              The single commanding piece behind the wordmark.
                        Landscape, dark, high impact. ~2400 x 1500 px.
  alina-portrait.jpg    One calm, editorial portrait. Portrait orientation,
                        ~1200 x 1500 px.

------------------------------------------------------------------
2) PORTFOLIO PIECES  →  assets/gallery/
------------------------------------------------------------------
  Vertical/portrait images read best. ~1000–1400 px on the long edge.
  Filenames map to styles — keep the names:

  Fine Line     fineline-01.jpg   fineline-02.jpg   fineline-03.jpg
  Blackwork     blackwork-01.jpg  blackwork-02.jpg  blackwork-03.jpg
  Botanical     botanical-01.jpg  botanical-02.jpg
  Lettering     lettering-01.jpg  lettering-02.jpg
  Ornamental    ornamental-01.jpg ornamental-02.jpg

  The "Flash / Drops" section reuses:
    fineline-02.jpg, blackwork-01.jpg, fineline-03.jpg

------------------------------------------------------------------
TIPS
------------------------------------------------------------------
- Compress before uploading (TinyPNG / Squoosh). The grain + motion stay
  fast only if the images are light. Aim < 300 KB each.
- Shoot on neutral skin tones against clean backgrounds.
- To add/rename/reorder pieces, edit the PIECES list at the top of
  assets/js/main.js — title, category and filename live there.
